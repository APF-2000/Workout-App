# Workout Log Web App

Static offline-friendly workout tracker for iPhone.

## What it does

- Tracks workouts, exercises, weights, and reps
- Shows history and progress charts
- Stores data in browser local storage
- Supports export and import for backups
- Can be installed to the iPhone home screen as a PWA

## How to use it

1. Host the `webapp` folder on any static host.
2. Open the site in Safari on your iPhone.
3. Use Share > Add to Home Screen.
4. Open the installed app from your home screen.

## Storage

- Day-to-day data is stored locally in your browser.
- Export backups regularly if you want long-term safety across devices or reinstalls.
